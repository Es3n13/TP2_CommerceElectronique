## 📝 Lessons Learned

### Stripe Payment Flow
- PaymentIntent requires PaymentMethod for complete payment
- Without PaymentMethod: Status = "requires_payment_method"
- With PaymentMethod: Status = "succeeded" (immediate)
- Frontend would normally call Stripe.js to create payment method tokens

### Service Communication
- All services must be running for integration testing
- PUT endpoint must exist for status updates
- Error handling (connection refused, 404) must be documented

### Environment Variables
- ASP.NET Core auto-loads environment variables by name
- Double underscore `__` maps to configuration hierarchy (`Stripe:SecretKey`)
- This approach works for both development and production
